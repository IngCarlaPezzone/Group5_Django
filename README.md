## Installation 

Clone the repo

```bash
git clone https://github.com/LucasAugusto-py/Group5_Django.git
```

Create a virtual env
```bash
py -m venv .venv
```
Activate the virtual env

```bash
.\venv\Scripts\venv
```
Install the dependences

```bash
pip install -r requirements.txt
```
